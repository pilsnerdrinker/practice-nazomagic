# P1なぞなぞマジック（真偽AoE判断）

FINAL FANTASY XIV 絶妖星乱舞のP1なぞなぞマジック（真偽AoE判断）を練習するための、非公式ファンメイド練習ツールです。

## 概要

このアプリでは、Round 1〜3 の真偽AoE系ギミックを個別に練習できます。

* Round 1：集合/散会 + 氷床
* Round 2：雷 + 氷床
* Round 3：集合/散会 + 雷 + 視線

各Roundには、判断のみを練習する「ノーマル」と、実際の立ち位置まで選ぶ「ハード」があります。

## 基本ルール

中央のボス付近に、上下2つの輪が表示されます。

* 上の赤い輪：Roundごとの上段ギミック
* 下の青い輪：Roundごとの下段ギミック

輪の中の球に `？` がない場合は、見たままの攻撃が来ます。
輪の中の球に `？` がある場合は、表示と逆の攻撃が来ます。

## Round 1

### ノーマル

Round 1 は、集合/散会と氷床を判断します。

上の赤い輪は、頭割りまたは散会を表します。

* `？` なし：表示どおり
* `？` あり：頭割りと散会が反転

アプリ上の回答では、頭割りを「集合」と表記しています。

下の青い輪は、氷床を表します。

* `？` なし：表示されている氷予兆は危険なので踏まない
* `？` あり：表示されている氷予兆は嘘なので踏む

### ハード

Round 1 ハードでは、自分のポジションを選び、実際の立ち位置をクリックします。

散会時は、実際の安置が 1・3 か 2・4 かによって散会位置が変わります。

集合時は、実際の安置内で以下のように集合します。

* タンヒラ：西側の安置内側
* DPS：東側の安置内側

## Round 2

### ノーマル

Round 2 は、雷と氷床を判断します。

上の赤い輪は雷を表します。

* `？` なし：表示されている雷予兆は危険なので踏まない
* `？` あり：表示されている雷予兆は嘘なので踏む

下の青い輪は氷床を表します。

* `？` なし：表示されている氷予兆は危険なので踏まない
* `？` あり：表示されている氷予兆は嘘なので踏む

### ハード

Round 2 ハードでは、雷と氷床の両方を避けられる位置をクリックします。

正誤判定は、クリックした位置が以下の両方を満たすかどうかで行っています。

* 実際の雷安地に入っている
* 実際の氷安地に入っている

条件を満たしていれば、盤面上のどこをクリックしても正解です。

## Round 3

### ノーマル

Round 3 は、集合/散会、雷、北の視線を判断します。

上の赤い輪は、頭割りまたは散会を表します。

* `？` なし：表示どおり
* `？` あり：頭割りと散会が反転

下の青い輪は雷を表します。

* `？` なし：表示されている雷予兆は危険なので踏まない
* `？` あり：表示されている雷予兆は嘘なので踏む

北の視線は以下のように処理します。

* 紫の目：北を見ない
* 黄色の？目：北を見る

### ハード

Round 3 ハードでは、自分のポジションを選び、実際の立ち位置と視線処理を回答します。

集合時は、雷の安置内で以下のように集合します。

* タンヒラ：北寄り
* DPS：南寄り

散会時は、雷の安置内で以下の並びになります。

* 北側：D3 / MT / D1 / H1
* 南側：H2 / D2 / ST / D4

視線処理も同時に判定します。

* 紫の目：見ない
* 黄色の？目：見る

## 正誤判定について

このアプリでは、表示された予兆そのものではなく、`？` による反転後の実際の攻撃をもとに正誤判定しています。

たとえば、雷の輪に `？` がない場合は、表示されている雷予兆を踏まない位置が正解です。
雷の輪に `？` がある場合は、表示されている雷予兆が嘘になるため、雷予兆上の位置が正解です。
